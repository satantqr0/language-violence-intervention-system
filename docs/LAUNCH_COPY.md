## 中文社区正文

```md
我开源了一个端侧 AI 语言暴力干预系统：

Language Violence Intervention System

它主要解决一个问题：很多语言伤害不是一次性爆发，而是长期反复出现的否定、羞辱、威胁、讽刺和情绪失控。这个系统希望把这些高风险互动尽早“看见、提醒、记录、复盘”。

目前支持：

- Raspberry Pi 5、Mac Mini、普通 x86_64 PC 部署
- 本地 Whisper 语音识别
- 规则引擎 + LLM 混合语义风险检测
- Web 管理控制台
- 温和 TTS 主动干预提醒
- 事件日志、趋势报告、人工标注
- 默认不保存原始录音，强调端侧和隐私保护

项目不是医疗诊断、法律证据或紧急救援替代系统，只是一个辅助观察和提醒工具。

GitHub：
https://github.com/satantqr0/language-violence-intervention-system

欢迎大家试用、提 Issue，也欢迎对隐私保护、误报控制、家庭/教育场景规则设计提出建议。
```

## English Launch Copy

```md
I built an open-source privacy-first edge AI system for verbal abuse detection and gentle intervention.

It runs on Raspberry Pi 5, Mac Mini, or a regular x86_64 PC. The system uses local speech recognition, acoustic risk analysis, rule-based pre-checks, LLM semantic analysis, and a protected Web UI to detect high-risk communication patterns and trigger non-aggressive voice reminders.

Key features:

- Local-first speech recognition with Whisper
- Rule engine + LLM semantic risk detection
- Gentle TTS intervention prompts
- Web dashboard for events, trends, review, and safety notes
- Raspberry Pi deployment scripts
- Privacy-focused design: no raw audio persistence by default

This is not a medical, legal, or emergency response system. It is intended as an assistive tool for observation, reflection, and human review.

GitHub:
https://github.com/satantqr0/language-violence-intervention-system

Feedback, issues, and privacy/safety suggestions are welcome.
```
