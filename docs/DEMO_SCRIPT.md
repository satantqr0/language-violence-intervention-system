# 60 秒演示脚本 / 60-Second Demo Script

## 中文版

**0-10 秒：项目简介**

这是一个隐私优先的端侧 AI 语言暴力识别与主动干预系统，可运行在 Raspberry Pi 5、Mac 或普通 PC 上。

**10-25 秒：WebUI 展示**

打开 Web 控制台，展示设备状态、检测趋势、运行模式、人员画像、安全处置和事件列表。

**25-40 秒：识别与提醒流程**

展示一条匿名演示事件：系统识别高风险表达和喊叫声学特征，随后通过 TTS 播放温和提醒。

**40-50 秒：隐私设计**

说明默认不保存原始录音，声纹模板、事件日志、人员画像、筛查结果和安全处置记录均保存在本机。

**50-60 秒：开源邀请**

项目已经开源，欢迎大家测试 Raspberry Pi 部署、反馈误报场景、补充干预话术和隐私改进建议。

## English Version

**0-10s: Intro**

This is a privacy-first edge AI system for verbal abuse detection and gentle
intervention. It runs on Raspberry Pi 5, Mac, or a regular PC.

**10-25s: WebUI**

Show the Web console: device status, trend chart, runtime modes, speaker
profiles, safety-case review, and event list.

**25-40s: Detection And Reminder**

Show an anonymous demo event: the system detects high-risk expression and
shouting-like acoustic risk, then plays a gentle TTS reminder.

**40-50s: Privacy**

Explain that raw recordings are not saved by default, and voiceprint templates,
event logs, speaker profiles, screening summaries, and safety-case records stay
on the local device.

**50-60s: Open Source Call**

The project is open source. Feedback on Raspberry Pi deployment, false
positives, intervention prompts, and privacy improvements is welcome.
